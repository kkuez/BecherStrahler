Uses jwiringpi lib by soonuse!

RPI:
Client anpingen
Zur Sicherheit in /etc/hosts eigene IP eintragen

Starten mit:
java -Djavafx.platform=gtk2 -jar MyJavafxApp.jar
(Greift auf ein alternatives, schnelleres JavaFX Framework zu)

In Console sind folgende Modi möglich, einfach eingeben wenn das Programm läuft:

1:	Setze Alle Pins dauerhaft auf HIGH
2:	Setze Alle Pins dauerhaft auf LOW
3:	Init Main neu
4:	Starte JFX GUI
5:  Kalibrierung
